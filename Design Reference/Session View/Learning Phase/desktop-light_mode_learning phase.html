What Are React Components?

Components are the building blocks of any React application. Think of them as custom, reusable HTML elements that contain both the logic and the structure of a specific part of your interface.

Every component follows a simple mental model: it accepts inputs (called "props") and returns a description of what should appear on the screen using JSX. Below is a basic functional component example. 

function Greeting() {
  return <h1>Hello, Skill Master!</h1>;
}

By breaking your UI into independent components, you make your code easier to maintain, test, and share across different parts of your roadmap.
